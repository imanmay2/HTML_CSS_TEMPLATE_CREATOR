# HTML_CSS_TEMPLATE_CREATOR
THIS IS A PYTHON PROJECT WHICH IS DESIGNED TO PROVIDE A HTML_CSS TEMPLATE, AS PER USER'S REQUIREMENTS.


## Usage

- Open any editor like  VISUAL STUDIO CODE, run the above python code.
